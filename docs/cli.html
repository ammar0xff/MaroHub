CLI Reference

CLI Commands

  • maro search <query> — Search for games by name or keyword.
  • maro list — List all available games.
  • maro download <game-id> — Download a game by its ID (magnet link).
  • maro info <game-id> — Show detailed info for a game.
  • maro update — Update the local games database.
  • maro help — Show help and usage info.

Examples

  • maro search doom — Find all games with "doom" in the name.
  • maro download 42 — Download the game with ID 42.
  • maro info 42 — Show details for game 42.
